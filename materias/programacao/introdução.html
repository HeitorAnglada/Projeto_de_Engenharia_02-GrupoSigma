Básico de Programação

Lógica de programação

A principal ideia ao se programar é que o computador entende os comandos de maneira sequencial, ou seja, o computador realizará os comandos de cima para baixo. Assim, tomando como exemplo o código abaixo:

{
int A, B;
A = 1;
B = A + 3;
printf(“%d”, B);
}

O computador primeiramente criará duas variáveis do tipo inteiro A e B, reservando um espaço da memória para comportar seus valores. Então depois disso, ele armazenará o valor 1 na variável A, alocando esse valor no espaço reservado. Então depois, vai alocar na variável B o valor de A + 3. Por fim, mostrará o valor de B.

Além do tipo inteiro, as variáveis também pode ser do tipo float, o qual corresponde a todos os números reais, do tipo boolean, o qual serve principalmente para armazenar estados verdadeiro ou falso, e char, o qual serve para armazenar caracteres em vez de valores.

As funções servem para relacionar as variáveis de determinada forma, no caso do printf, ela serve para mostrar conteúdo na tela, seja valores de variáveis, seja informações qualquer. Desse modo, já que printf é considerado uma função de saída de informação, scanf é considerado uma função de entrada de informação, ou seja, ao usar o scanf, é alocado na posição de memória correspondente a variável desejada, o valor a ser inserido.

Além disso, temos estruturas de repetição, as quais permitem que um determinado trecho do código seja repetido de acordo com as condições pré estabelecidas. São elas: for, while e do-while. Além dessas também há as estruturas de condicionamento, que realizam determinada parte do código caso alguma condição seja atendida. São elas: if, else e switch.
